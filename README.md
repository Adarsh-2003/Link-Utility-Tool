# Link Utility Tool

Small static page that fixes **broken URLs** (extra spaces or line breaks from copying comments, OCR, etc.). You get a working link, optional copy, and a clickable result.

## How to use

1. Open `index.html` in your browser (double-click or drag into a tab).
2. Paste a messy URL — it cleans on paste, or use **Clean link**.
3. Open the link or use **Copy clean link**.

Plain **HTML**, **CSS**, and **JS** — no build step.

## Files

| File        | Purpose              |
| ----------- | -------------------- |
| `index.html` | Page structure      |
| `styles.css` | Layout and theme    |
| `script.js`  | Clean URL + copy    |

**Note:** Copy to clipboard works best over **http(s)** or **localhost**. Some browsers limit it on `file://`.
